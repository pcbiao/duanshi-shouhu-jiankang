# 断食守护健康

这个目录用于存放“断食守护健康”的安卓版/PWA 文件。

当前保留两种形态：

- 多文件版：用于日常开发和 PWA 运行。
- 单文件测试版：用于快速打开测试或临时分发。

文件说明：

- `index.html`：多文件入口
- `styles.css`：页面样式
- `script.js`：功能逻辑
- `manifest.webmanifest`：PWA 配置
- `sw.js`：PWA 离线缓存
- `icon.svg`：图标
- `reminder.mp3`：多文件版稳糖提醒铃声
- `断食守护健康-v2.2.1.html`：单文件测试版，已内嵌稳糖提醒铃声
- `archive/`：旧版或方案预览归档

主目录里的原版文件不受这个目录影响。
